1.1 Styles and Themes

Word Styles are predefined sets of formatting that you can apply to text instantly. Using Styles ensures consistency throughout your document.

Built-in Styles

  • Heading 1, 2, 3 - For document hierarchy
  • Normal - Standard body text
  • Title - Document title formatting
  • Subtitle - For document subtitles
  • Quote - Emphasized quotations

Applying and Modifying Styles

  1. Select text you want to format
  2. In Home tab, find Styles group
  3. Click desired style or dropdown arrow for more options
  4. Right-click style to modify formatting

1.2 Tables of Contents and Bookmarks

A Table of Contents (TOC) is automatically generated from heading styles and helps readers navigate long documents.

Creating a Table of Contents

  1. Format document headings with proper styles (Heading 1, 2, etc.)
  2. Place cursor where you want TOC to appear (usually after title)
  3. Go to References tab > Table of Contents
  4. Choose automatic or manual table of contents style
  5. Update TOC after making changes to headings

Bookmarks create invisible markers in your document that you can link to. This is useful for creating cross-references.

1.3 Headers, Footers & Page Numbers

Headers and footers appear on every page and are ideal for adding document titles, dates, page numbers, or company information.

Inserting Headers and Footers

  1. Go to Insert tab
  2. Click Header or Footer
  3. Choose a design or click 'Edit Header/Footer'
  4. Type your content (you can use placeholders for automatic fields)
  5. Click 'Close Header and Footer' when done

Chapter Summary

  • Styles ensure consistent formatting across documents
  • Table of Contents automatically generated from heading styles
  • Headers and Footers add professional touches to every page
