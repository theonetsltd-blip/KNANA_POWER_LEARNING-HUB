12.1 Understanding the Internet

The Internet is a global network of connected computers that share information.

Key Terms:

  • IP Address - Unique identifier for computers on internet
  • HTTP/HTTPS - Protocol for transferring web pages
  • Web Browser - Software to view websites
  • Website - Collection of related web pages
