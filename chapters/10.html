10.1 The Word Interface

Microsoft Word is a word processing application for creating documents.

Main Components:

  • Title Bar - Shows document name
  • Ribbon - Contains formatting and editing tools
  • Document Area - Where you type content
  • Ruler - Sets margins and indents
  • Status Bar - Shows page count and zoom

10.2 Creating and Saving Documents

Steps to Create:

  1. Open Microsoft Word
  2. Select 'Blank Document'
  3. Start typing content
  4. Click File > Save
  5. Enter document name
  6. Choose location and click Save
