Key Concepts & Glossary

Operating System (OS)

The software that manages computer hardware and lets users run applications

Desktop

The main screen you see when the OS starts - shows icons, taskbar, and wallpaper

Taskbar

The bar at the bottom (or side) of the screen showing open programs and system info

File

A collection of data stored on the computer with a name and specific location

Folder/Directory

A container that holds files and other folders to organize data

Icon

A small picture representing a program, file, or folder

3.1 What is an Operating System?

An Operating System (OS) is a software program that acts as a bridge between you and the computer hardware.

üí° Think of it this way:

The OS is like a manager in a hotel:

  • Guests (users) tell the manager (OS) what they want
  • The manager coordinates with staff (hardware) to fulfill requests
  • The manager ensures everyone shares resources fairly

Main Functions of an OS:

  1. Control Hardware - Manage CPU, RAM, storage, printer, screen
  2. Run Programs - Let you open and use applications
  3. Manage Files - Organize and protect your documents and data
  4. Provide User Interface - Create the desktop, menus, and icons so you can interact

3.4 Using the Start Menu & Taskbar

Opening Programs from Start Menu:

  1. Click the 'Start' button (Windows logo) at bottom left
  2. You'll see categories like All Apps, Recent, Recommended
  3. Find the program you want and click it
  4. The program will open in a new window

Managing Windows:

Minimize: Hide window but keep program running (click - button)

Maximize: Make window full screen (click [] button)

Restore: Return to smaller window size (click [] button again)

Close: Shut down the program (click X button)

Taskbar Features:

  • Shows all open programs with thumbnails when you hover
  • Click a program on taskbar to bring its window to front
  • Right-click taskbar for options like cascade, tile, lock

3.5 File Explorer & Basic Navigation

File Explorer is the program that shows your files and folders. To open it:

  • Click Start menu > type 'File Explorer' > press Enter
  • Or press Windows key + E

Key File Explorer Areas:

üîç Search Bar (Top)

Type to find files or folders by name

üìÅ Left Panel (Navigation)

Shows quick access to common folders: This PC, Documents, Downloads, Pictures

üìÇ Main Window (Center)

Shows files and folders in current location. Double-click folder to enter, double-click file to open

üìä Details Pane (Right, optional)

Shows information about selected file: size, date modified, type

üí° Useful Tips:
