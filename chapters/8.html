8.2 Using USB Drives

Copying Files to USB:

  1. Plug USB into USB port on computer
  2. Open File Explorer, locate files to copy
  3. Right-click and select 'Copy'
  4. Open USB drive folder and paste
  5. Safely eject USB (right-click > Eject)

⚠️ WARNING: Always eject USB safely before removing to avoid data corruption

8.3 Cloud Storage Basics

Cloud storage stores files on remote servers accessible from anywhere with internet.

Popular Services:

  • Google Drive - 15GB free, part of Google account
  • OneDrive - 5GB free, part of Microsoft account
  • Dropbox - 2GB free, collaborative file sharing

Chapter Summary

  • Multiple storage options available for different needs
  • USB drives are portable and reliable backup
  • Cloud storage provides accessibility from anywhere
  • Regular backups prevent permanent data loss
