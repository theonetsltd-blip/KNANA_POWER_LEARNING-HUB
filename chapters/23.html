23.1 Common Computer Problems & Solutions

ProblemCauseSolution

Blue Screen ErrorSystem crashRestart, update drivers and OS

No InternetConnection lostRestart router, check cables

23.2 System Maintenance Tasks

Regular Maintenance:

  1. Update Windows and software
  2. Run antivirus scans
  3. Clean temporary files (Disk Cleanup)
  4. Defragment hard drive (Windows 7/8)
  5. Check disk space
  6. Backup important files

23.3 Managing Disk Space & Performance

Performance Tips:

  • Delete unnecessary files and programs
  • Disable startup programs (Startup tab in Task Manager)
  • Increase RAM if budget allows
  • Clear browser cache regularly
  • Use SSD instead of HDD if possible

Preventive Maintenance Checklist

  • ✓ Monthly: Update Windows, scan antivirus
  • ✓ Quarterly: Clean temp files, backup data
  • ✓ Yearly: Physical cleaning, check hardware
  • ✓ Always: Use strong passwords, safe browsing
