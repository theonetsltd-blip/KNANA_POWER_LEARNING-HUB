CHAPTER 20

Project Execution

Execute projects effectively combining document, spreadsheet and presentation skills

Duration
8 hours
Level
NQF 3
Credits
3 units

Learning Outcomes

  • ✓ Execute project tasks on schedule
  • ✓ Track project progress
  • ✓ Create project documentation
  • ✓ Manage project team communication
  • ✓ Handle project changes and issues

20.1 Executing Project Tasks

Project execution is putting the plan into action.

Key Activities:

  • Assign tasks to team members
  • Monitor task completion
  • Track time and resource usage
  • Document completed work
  • Report progress to stakeholders

20.2 Project Documentation

Important Documents:

DocumentPurpose
Project CharterAuthorizes the project
Requirements DocumentDetails what must be delivered
Status ReportsTracks progress and issues
Meeting MinutesRecords decisions and action items

20.3 Team Communication

Effective communication is crucial for project success.

Communication Tools:

  • Email - Formal communications, decisions
  • Meetings - Face-to-face discussions (in-person or video)
  • Project Management Tools - Trello, Asana for task updates
  • Status Reports - Scheduled progress updates

Handling Issues & Changes

  • ✓ Document all issues immediately
  • ✓ Assess impact on timeline and budget
  • ✓ Propose solutions to stakeholders
  • ✓ Update project plan accordingly
  • ✓ Communicate changes to team
