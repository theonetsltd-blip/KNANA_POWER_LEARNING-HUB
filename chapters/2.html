Key Concepts & Glossary

Hardware

The physical parts of a computer that you can touch - like keyboard, monitor, and circuit boards

CPU (Central Processing Unit)

The 'brain' of the computer that performs all calculations and controls all other parts

RAM (Random Access Memory)

Temporary memory that stores data currently being used. It is erased when computer is switched off

Storage Device

Permanent memory that keeps data even when powered off - like Hard Drive, SSD, or USB stick

Motherboard

The main circuit board that connects all hardware components together

Peripheral

Any external device connected to the computer, like printer, scanner, or mouse

2.1 Main Components of a Computer

Every computer has several main parts that work together:

Component Function Location
CPU Processes all information and controls all other parts Inside system unit
RAM Stores data that CPU is currently using - fast but temporary Inside system unit
Hard Drive Permanently stores all programs and files even when powered off Inside system unit
Motherboard Main circuit board connecting all components Inside system unit
Power Supply Supplies electrical power to all components Inside system unit
Monitor Displays the image and information from the computer Outside (desktop)

2.2 Understanding CPU, RAM, and Storage

ğŸ’¡ Think of it this way:

Your computer is like a restaurant kitchen:

CPU = Chef

The chef (CPU) follows recipes (programs) and prepares dishes (processes data)

RAM = Work Counter

The work counter (RAM) holds ingredients (data) the chef is currently using - temporary only

Hard Drive = Storage Room

The storage room (Hard Drive) keeps recipes and ingredients permanently

Motherboard = Kitchen Layout

The kitchen layout (Motherboard) connects all stations together

CPU Speed (Processor Speed)

CPUs are measured in GHz (Gigahertz). Higher GHz = faster processing:

  • 2 GHz: Slower, basic tasks
  • 3-4 GHz: Good for everyday use
  • 5+ GHz: Very fast, gaming and video editing

RAM Amount

More RAM means you can run more programs at once:

  • 2-4 GB: Basic browsing and office work
  • 8 GB: Good for everyday use with multiple programs
  • 16+ GB: Professional work, video editing, gaming

Storage Types

Hard Disk Drive (HDD)

Uses spinning disks to store data - slower but cheaper and larger capacity

Solid State Drive (SSD)

Uses flash memory - very fast but more expensive

Chapter Summary

  • Hardware is the physical part of a computer
  • CPU (brain) processes all information and controls all other parts
  • RAM is fast temporary memory for current tasks
  • Hard Drive/SSD permanently stores all programs and files
  • Motherboard connects all components together
  • Input devices send information to the computer (keyboard, mouse, scanner)
  • Output devices receive information from the computer (monitor, printer, speakers)
