6.1 Mouse Operations

üñ±Ô∏è Mouse Parts:

Left Button

Used for selecting, clicking, and dragging

Right Button

Opens context menu with options

Scroll Wheel

Moves page up/down; click to open links in new tab

Pointer

Arrow that moves when you move the mouse

üìã Basic Mouse Techniques:

Action How Used For
Click Press left button once Select items, press buttons
Double-Click Press left button twice quickly Open files, folders, programs
Right-Click Press right button once See options, copy, paste, rename
Drag Hold left button & move mouse Move files, highlight text
Scroll Roll scroll wheel up or down Move page up/down in long documents

üí° Pro Tip:

Double-click must be FAST (within 0.5 seconds). If slow, mouse will think you did two separate clicks!

6.2 Keyboard Basics

üìå Main Keyboard Sections:

Typing Keys

A-Z letters, 0-9 numbers, and symbols like !, @, #, $

Function Keys

F1-F12 - each program uses them differently

Modifier Keys

Shift, Ctrl, Alt - used with other keys for shortcuts

Navigation Keys

Arrow keys, Page Up/Down, Home, End

üî§ Important Special Keys:

Enter Confirms input, moves to new line, activates buttons
Backspace Deletes character before cursor
Delete Deletes character after cursor or selected items
Spacebar Creates space between words
Tab Moves cursor 5 spaces, or cycles through form fields
Escape (Esc) Cancels current action or closes dialog boxes

‚úÇÔ∏è Editing Shortcuts

üíæ File/Program Shortcuts

üíª Windows-Specific Shortcuts:

  • Alt + Tab - Switch between open programs

6.4 Proper Typing Technique

üñêÔ∏è Hand Position (QWERTY Keyboard):

üìç Home Row Keys:

Your fingers should rest on the home row:

A S D F J K L ;

Left hand (on ASDF): Fingers: Index on F, Middle on D, Ring on S, Pinky on A

Right hand (on JKL;): Fingers: Index on J, Middle on K, Ring on L, Pinky on ;

üìå Typing Tips:

  • Look at the paper, NOT the keyboard
  • Fingers stay on home row when not typing
  • Use correct finger for each key (builds muscle memory)
  • Space bar with thumb
  • Keep wrists straight, not bent
  • Take breaks to avoid RSI injury

üí™ Building Speed:

Accuracy first, speed comes later! It's better to type slowly and correctly than fast with many errors.
