14.1 Spreadsheet Structure

A spreadsheet is a grid of cells organized in rows and columns.

Key Components:

  • Cell - Individual box holding one value (A1, B2, etc.)
  • Row - Horizontal line of cells (numbered 1, 2, 3...)
  • Column - Vertical line of cells (labeled A, B, C...)
  • Worksheet - Individual sheet in a workbook

14.2 Creating Formulas

Formulas perform calculations on cell values.

Common Formulas:

FormulaPurposeExample
=SUM()Add numbers=SUM(A1:A10)
=AVERAGE()Calculate average=AVERAGE(B1:B5)
=COUNT()Count cells with numbers=COUNT(C1:C10)
=MAX()Find largest value=MAX(D1:D20)

14.3 Creating Charts

Charts visually represent data.

Steps to Create Chart:

  1. Select data range
  2. Click 'Insert' > 'Chart'
  3. Choose chart type (bar, pie, line)
  4. Click 'Next' to customize
  5. Click 'Finish' to insert

Chapter Summary

  • Spreadsheets organize and calculate data
  • Formulas automate calculations
  • Charts make data visualization clear
