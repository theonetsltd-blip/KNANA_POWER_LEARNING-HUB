13.1 Creating Tables

Tables organize data in rows and columns.

Steps to Insert Table:

  1. Click 'Insert' tab
  2. Click 'Table'
  3. Select number of rows and columns
  4. Click on grid to create table

13.2 Headers, Footers & Page Numbers

Headers appear at the top of each page, footers at the bottom.

Adding Headers:

  1. Click 'Insert' tab
  2. Click 'Header' > 'Edit Header'
  3. Type header text
  4. Close header editor

Adding Page Numbers:

  1. Click 'Insert' > 'Page Number'
  2. Choose position (top, bottom, etc.)
  3. Select number format

13.3 Styles and Formatting

Styles ensure consistent formatting throughout a document.

Style TypeUse
Heading 1Main document titles (Large, bold)
Heading 2Section titles (Medium)
NormalBody text (Standard formatting)
QuoteQuoted text (Indented, italics)
