5.2 Proper Shutdown Procedure

Never just turn off the power! Always use proper shutdown to avoid data loss.

üìã Steps to Properly Shut Down Windows:

  1. Click Start menu at the bottom left
  2. Click Power icon - looks like circle with line at top
  3. Select Shut down (or Shutdown from the menu)
  4. Wait for the screen to go black and computer to turn off completely

‚ö†Ô∏è NEVER DO THIS:

  • ‚ùå Don't yank the power cord
  • ‚ùå Don't force the power button off
  • ‚ùå Don't shut down while programs are saving files
  • ‚ùå Don't ignore 'Save changes?' prompts

‚ö° Alternative Shutdown Methods:

  • Keyboard shortcut: Press Alt + F4 on desktop (closes all programs first)
  • Command prompt: Type 'shutdown -s -t 0' (shuts down immediately)

5.3 Restart & Sleep Modes

üîÑ Restart (Reboot)

What it does: Shuts down OS and boots it up again

When to use: After installing updates, fixing issues, or freeing up RAM

Time: About 1-2 minutes

üò¥ Sleep Mode

What it does: Saves your work and turns off display and hard drive to save power

When to use: When taking a break (15 min to few hours)

Wake up: Move mouse or press any key

‚úì Always save and close work before sleeping! A power outage while sleeping can cause data loss.

5.4 Handling Problems & Crashes

‚ùå If the Screen Freezes:

  1. Wait 30 seconds - computer might be thinking
  2. Try: Ctrl + Alt + Delete (opens Task Manager)
  3. Find the frozen program and click 'End Task'
  4. If that doesn't work, hold Power button for 5 seconds (force shutdown)

üî¥ If the Computer Won't Start:

  • Check: Is power cord plugged in and outlet working?
  • Try: Pressing power button for 10 seconds
  • Check: Are all cables properly connected?
  • If still nothing: Call IT support or technician

‚ö° Blue Screen of Death (BSOD):

If you see a blue screen with white text, it's a serious Windows error:

  • Don't panic - computer is protecting itself
  • Let it restart automatically
  • If it keeps happening, report to IT support
