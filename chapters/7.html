7.1 What Are Files and Folders?

A FILE is a collection of data with a name stored on your computer. A FOLDER (also called directory) is a container that holds files and other folders.

File Extensions

File extensions are letters after the dot (.) that show what type of file it is:

  • .doc, .docx - Word document
  • .xls, .xlsx - Excel spreadsheet
  • .ppt, .pptx - PowerPoint presentation
  • .txt - Plain text file
  • .jpg, .png, .gif - Image files
  • .pdf - Portable Document Format
  • .zip - Compressed archive

7.2 Creating Folders

  1. Open File Explorer (Windows + E)
  2. Navigate to Documents folder
  3. Right-click in empty space
  4. Select 'New > Folder'
  5. Type a descriptive name (no special characters)
  6. Press Enter

Naming Folders

  • Use clear, descriptive names
  • Use dates in YYYY-MM-DD format (2024-01-15)
  • Avoid spaces - use underscores or hyphens
  • Examples: Documents_2024, Projects, Invoices_Jan-2024
